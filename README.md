# TheMealPrepApp 🥗

## What is it about? ❓

Recipe app in which you can search by name or filter by area on a list. Contains a detailed view of each recipe with name, image, written recipe and video of it.

## What can you find? 🔎

- Adequate structure of files and folders.

- Clean architecture.

- Repository pattern.

- MVVM pattern.

- Testing.

- Accessibility.
  
- Animations.

## Build with 🛠️

- SwiftUI
- Combine
- Async await
- Webkit
